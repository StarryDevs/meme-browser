# meme-browser

meme-browser 是一个用于浏览 Meme 的浏览器应用。它支持 Meme 的搜索和展示，方便用户快速查找各种 Meme。

## 主要功能

- Meme 浏览与分类
- Meme 搜索功能
- 多种图片格式支持

## 安装方法

```bash
git clone <仓库地址>
cd meme-browser
./gradlew build
```

## 使用方法

```bash
./gradlew run
```

启动后，访问 `http://localhost:<端口号>`，即可使用 meme-browser 服务。

## 目录结构

```text
src/           # 后端代码
src-vue/        # 前端代码

```
